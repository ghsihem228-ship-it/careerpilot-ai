## 📌 How It Works

1. Upload your resume PDF  
2. Paste a job description  
3. Get ATS score and resume level  
4. View detected skills and improvement suggestions  
5. Check job match score and missing keywords  
6. Generate interview questions  
7. Generate a learning roadmap  
8. Estimate salary ranges  
9. Generate a tailored cover letter
